# Implementations

논문, 강의, 개념 노트에서 배운 내용을 작은 코드로 구현해보는 공간입니다.

완성된 프로젝트가 아니어도 괜찮습니다. 중요한 것은 읽은 내용을 코드로 옮기면서 무엇을 이해했고 무엇이 어려웠는지를 기록하는 것입니다.

## Implementation Format

```text
Topic:
Source:
Goal:
Input:
Method:
Output:
What worked:
What failed:
Next step:
```

## Rule

- 처음에는 작게 구현합니다.
- 결과보다 구현 과정과 배운 점을 남깁니다.
- 재사용 가능한 프로젝트 코드는 [`academic-projects`](https://github.com/minbro-01/academic-projects)의 해당 프로젝트 폴더로 옮깁니다.
- 외부 코드를 참고한 경우 출처와 내가 수정한 부분을 구분합니다.
