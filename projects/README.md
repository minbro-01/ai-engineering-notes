# Projects

이 폴더는 AI Engineering을 공부하면서 진행하는 실험과 구현을 프로젝트 단위로 정리하는 공간입니다.

각 프로젝트는 하나의 질문이나 학습 목표에서 시작합니다. 예를 들어 `분류 모델의 baseline을 어떻게 만들고 평가할까?` 같은 질문을 하나의 프로젝트로 다룹니다.

## Project Naming

```text
01-project-name/
02-project-name/
03-project-name/
```

번호는 학습 순서를 나타냅니다. 프로젝트가 완성되지 않아도, 공부 과정과 시행착오를 남길 수 있습니다.

## Project Template

```text
project-name/
├─ README.md      # 프로젝트 개요
├─ src/           # 코드
├─ notebooks/     # 실험 노트북
├─ data/          # 데이터 설명 또는 샘플
├─ results/       # 결과물
├─ report.md      # 결과 정리
└─ references.md  # 참고 자료
```
