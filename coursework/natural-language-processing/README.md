# Natural Language Processing

기본 NLP부터 LLM까지 공부한 내용과 DRT 운행 정책 제안 프로젝트를 정리하는 공간입니다.

## Topics

| Topic | What to understand |
| --- | --- |
| Text preprocessing | tokenization, normalization, vocabulary |
| Word representation | BoW, TF-IDF, embedding |
| Sequence models | RNN, LSTM, GRU 기반 텍스트 처리 |
| Attention / Transformer | self-attention과 Transformer 구조 |
| LLM | LLM의 기본 사용 방식과 한계 |
| Policy recommendation | 텍스트 기반 분석을 정책 제안으로 연결하는 흐름 |

## Project Connection

| Project | Description |
