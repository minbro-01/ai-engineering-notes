# Reinforcement Learning

강화학습 수업에서 배운 MDP, DP, TD, Q-learning, SARSA, Deep RL 개념을 정리하는 공간입니다.

## Topics

| Topic | What to understand |
| --- | --- |
| MDP | state, action, reward, transition, policy |
| Dynamic Programming | policy evaluation, policy iteration, value iteration |
| TD Learning | Monte Carlo와 TD 방식의 차이 |
| Q-learning | off-policy value learning |
| SARSA | on-policy value learning |
| Deep RL | deep network를 value/policy approximation에 사용하는 방식 |

## Study Questions

- MDP로 문제를 정의한다는 것은 무엇인가?
- DP, TD, Q-learning, SARSA는 어떤 가정과 차이를 가지는가?
- Deep RL은 기존 강화학습에서 어떤 부분을 확장하는가?
