# Data Structures & Algorithms

자료구조와 알고리즘 구현 연습을 정리하는 공간입니다.

AI Engineering과 직접적으로 떨어져 보일 수 있지만, 모델 구현과 문제 해결을 안정적으로 하기 위한 기본기라고 보고 정리합니다.

## Topics

| Topic | What to practice |
| --- | --- |
| List | 배열과 연결 리스트의 차이, 삽입/삭제/탐색 |
| Stack | LIFO 구조와 괄호 검사, DFS 응용 |
| Queue | FIFO 구조와 BFS 응용 |
| Heap | 우선순위 큐와 정렬/탐색 문제 |
| Tree | 트리 순회, 이진 트리, 탐색 구조 |
| Graph | BFS, DFS, shortest path 기초 |

## Practice Format

```text
Problem:
Data structure:
Algorithm idea:
Implementation:
Complexity:
What I learned:
```
