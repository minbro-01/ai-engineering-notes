# Experiment Logs

AI Engineering 실험의 조건, 결과, 실패 원인, 다음 시도를 기록하는 공간입니다.

실험 로그는 결과가 좋을 때만 남기는 것이 아니라, 실패했거나 예상과 달랐던 경우에도 남깁니다.

## Log Format

```text
Date:
Experiment:
Question:
Dataset:
Model:
Setting:
Result:
Observation:
Problem:
Next experiment:
```

## Rule

- 실험 조건을 결과와 함께 기록합니다.
- 비교할 수 있도록 metric과 setting을 분리해서 적습니다.
- 다음 실험으로 이어지는 질문을 남깁니다.
